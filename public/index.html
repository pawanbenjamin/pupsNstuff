Welcome to Pups 'n Stuff

A simple CRUD API built with Express and PG to serve all your CRUD-ding needs

Endpoints:

GET /api/puppies

GET /api/puppies/:id

POST /api/puppies

PATCH /api/puppies/:id

DELETE /api/puppies/:id

GET /api/owners

GET /api/owners/:id

PATCH /api/owners/:id

DELETE /api/owners/:id

GET /api/tricks

GET /api/ticks/:id

POST /api/tricks

PATCH /api/tricks/:id

DELETE /api/tricks/:id

POST /api/puppies_tricks/:puppyId/:trickId

DELETE /api/puppies_tricks/:puppyId/:trickId
